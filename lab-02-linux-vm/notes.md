# Lab 2 - Linux Virtual Machine
** What i did:** Deployed an Ubuntu Server 24.04 LTS virtual machine using the B2a size (free tier eligible). Connected to it remotely using SSH through Azure Cloud Shell. Ran uname -a to confirm i was inside the VM. Stopped and deallocated the VM when finished to avoid charge

** What i learned:** Virtual machines are IaaS, you maange the OS and everything above it, Azure manages the hardware. Stopped (deallocated) means the vm is off and you are not charged for compute.

** Services used:** Azure Virtual Machines, Cloud Shell, SSH
