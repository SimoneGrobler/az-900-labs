# Lab 8 - Azure Cost Management & Budget

**What i did:** Used Azure Cost Management to analyse subcription spending broken down by resource and service. Created a $10 monthly budget with two alert conditions. Exported a cost report as an Excel file showing itemised charges.

**What i learned:** Cost management is one the the most critical skills for any cloud role, uncontrolled spending is a common problem in organisations.

**Services used:** Azure Cost Management + Billing, Cost Analysis, Budgets, Exports.
