# Lab 7 - Azure Monitor & Alerts

**What i did:** Used Azure Monitor to view CPU percentage metrics for a virtual machine over a 24-hour period. Created a metric alert rule that fires when average CPU exceeds 80% over a 5-minute window. Created and Action Group with email notification linked to the alert. verified the alert rule was active in the ALert Rules list.

**What i learned:** Azure Monitor is the central hub for all monitoring, metrics, and alerting across an entire Azure subscription. Action Groups define exactly what happens when an alerts fire. Critical in a real cloud role to catch a problem before users are affected, to save time, money and reputation.

**Services used:** Azure Monitor, Metrics Explorer, Alert Rules, Action Grous.
